盘前交易计划制定规则

为规范交易行为，消除交易过程中的贪婪与恐惧情绪，确保每一次入场有清晰理由、每一次出场有明确界限，特制定本盘前交易计划规则。本规则核心在于通过自我反思、精准研判市场，建立可执行、可追溯的交易逻辑，核心需围绕以下8个关键问题展开，逐一明确答案后，方可执行后续交易操作。

一、核心反思问题及执行细则

（一）当前市场正在干什么

核心目的：明确当前市场整体趋势方向，避免在趋势模糊时盲目入场，为后续交易决策奠定基础。

操作细则：若无法直观判断市场趋势，可借助EMA（指数移动平均线）快、中、慢三条均线辅助判断，具体判断标准如下：当快线（短期EMA）位于最上方，中线（中期EMA）位于中间，慢线（长期EMA）位于最下方时，即为明确的多头排列，对应市场处于多头趋势；反之，若慢线在上、中线居中、快线在下，则为空头排列，对应市场处于空头趋势；若三条均线相互缠绕、无明显上下顺序，则为震荡趋势，暂不建议优先入场。

（二）趋势的起点在哪里

核心目的：明确趋势级别，界定自身计划捕捉的利润区间，避免因趋势分析范围过宽、逻辑混乱，导致交易决策失误。

操作细则：为简化分析逻辑、聚焦核心趋势，需将分析的K线数量控制在250根以内（避免冗余数据干扰判断）；以市场极值点（阶段性最高位或最低位）为分割点，拆分K线走势，分割后仅关注极值点右侧的趋势走势，忽略左侧历史走势，重点锁定当前可捕捉的趋势段，明确自身计划获取的利润区间。

（三）当前趋势是什么阶段

核心目的：筛选有效交易机会，趋势末期风险极高，禁止开仓；仅当趋势处于非末期时，方可择机入场，降低交易亏损概率。

判断方法（借助MACD技术指标）：

1.  先找出MACD线在零轴上方出现死叉之前的波峰高点（即MACD线上升过程中的阶段性峰值）；

2.  找到该MACD波峰高点对应的K线极值高点（注意：K线极值高点与MACD波峰高点可能存在小幅偏差，以K线实际极值点为准）；

3.  持续观察当前趋势的所有波峰高点、K线极值点高点，若两者均呈现“不断抬高”的态势，则可判定当前右侧上涨趋势处于非末期，属于可追随的有效趋势；若波峰高点、K线极值点高点出现回落或持平，则大概率处于趋势末期，禁止开仓。

（四）当前趋势的级别是什么

核心目的：明确趋势强度与可操作空间，仅当本级别趋势获得更大级别趋势支撑时，才值得追随；小级别趋势波动节奏快，难以把控止盈止损，不建议优先操作。

判断方法（结合中枢与MACD指标）：

1.  中枢定义：市场交易中的密集成交区，是判断趋势级别的核心依据；

2.  筛选标准：找出当前趋势中所有密集成交区（假设为A、B、C三个区域），重点观察各区域对应的MACD回踩零轴的动作；

3.  级别判定：

 - 若某区域MACD刚好回踩零轴，即为标准本级别中枢；

 - 若某区域MACD回踩零轴且小幅跌破零轴，仍可视为本级别中枢，且级别大于仅回踩零轴的中枢；

 - 若某区域MACD仅有回踩动作，但未接近零轴，则属于更小级别的中枢；

4.  结论：当前盘面上，MACD回踩零轴动作最明显、级别最大的中枢，对应当前趋势的核心级别；若本级别趋势与该最大级别中枢对应的趋势一致，则该趋势值得追随，可规划入场操作。

（五）你的偏见是什么

核心目的：规避主观情绪对交易决策的干扰，明确自身交易倾向，确保偏见与市场实际趋势、自身交易逻辑一致，不盲目反向操作。

操作细则：本问题为前4个问题的汇总结论，需基于对市场趋势、级别、阶段的研判，明确自身主观交易倾向。示例：若研判得出当前市场处于本级别上涨趋势，且趋势处于非末期，则自身交易偏见应明确为“逢低做多”，所有交易操作需围绕该偏见展开，避免因临时情绪波动，做出做空等反向操作。

（六）你是顺哪个大级别，逆哪个小级别周期

核心目的：遵循市场惯性规律，趋势一旦形成不会轻易改变，通过“顺大逆小”的周期搭配，兼顾利润空间与入场时机，降低交易风险。

操作细则：不局限于当前观察的趋势周期，重点关注周期间的联动关系：优先追随大级别周期趋势（建议以1小时周期为核心大级别），依托大级别趋势的惯性，把握整体利润空间；在小级别周期（建议以15分钟周期为核心小级别）中寻找反转机会，择机入场（即顺1小时大级别趋势，逆15分钟小级别周期反转入场），实现“大趋势保利润、小周期抓时机”的平衡。

（七）你的防守位在哪里

核心目的：设定明确的风险控制边界，提前锁定最大亏损，避免因市场突发波动导致亏损扩大，守住交易本金安全。

操作细则：防守位设定需结合前文研判结果，核心参考依据包括：当前趋势的极值点、最大级别中枢的边缘位置、EMA慢线支撑/压力位；防守位一旦设定，需严格执行，若市场价格触及防守位，立即离场，不抱侥幸心理，不随意调整防守位（除非趋势级别、阶段发生明确变化，且重新研判后调整有明确依据）。

（八）要不要到交易周期里去

核心目的：确认当前交易时机是否成熟，避免在非交易周期内盲目入场，确保每一次入场都符合自身交易逻辑，与盘前计划高度匹配。

操作细则：结合前7个问题的答案，综合判断：若前7个问题均有明确、一致的答案（趋势方向、级别、阶段清晰，偏见明确，防守位合理，周期搭配得当），且小级别周期出现符合预期的反转信号，则可进入交易周期，执行入场操作；若任意一个问题答案模糊、逻辑矛盾，或小级别周期未出现有效反转信号，则暂不进入交易周期，继续等待，直至所有条件满足。

二、总则要求

1.  盘前计划需在每一次入场前完成，未完成8个问题的反思、未明确所有答案前，禁止任何入场操作；

2.  反思过程需客观、理性，杜绝主观臆断，所有判断需有明确的技术指标（EMA、MACD）、K线走势、中枢分析作为依据，不凭感觉决策；

3.  严格遵循“趋势末期不开仓、无防守位不开仓、交易周期不明确不开仓”的三大原则，坚守风险控制底线；

4.  每次交易后，需结合盘前计划进行复盘，核对入场、出场是否符合计划要求，反思问题研判过程中的不足，优化后续盘前计划制定逻辑。

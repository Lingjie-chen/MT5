# 量化策略变更日志生成器

将技术性 git 提交转化为清晰的策略迭代文档。

## 适用场景

- 发布新版本策略时生成 Release Notes
- 跟踪策略参数（Risk Tier、SL/TP 规则）的变更历史
- 记录 Quantum Engine 或风控模块的更新
- 生成周/月策略迭代报告
- 向团队同步策略变更

## 使用方式

```
从最近一次 release 以来的提交生成变更日志
```

```
生成过去一周的策略变更记录
```

```
生成 v2.0.0 到 v3.0.0 之间的变更日志
```

## 变更分类体系

本 Skill 会将所有 git 提交自动归类为以下量化交易专属类别：

| 类别 | 图标 | 包含内容 |
|------|------|---------|
| 策略更新 | 📈 | 入场/出场规则变更、SMC 参数调整、Grid 策略优化 |
| 风控调整 | 🛡️ | Risk Tier 变更、回撤阈值、保证金规则 |
| 引擎优化 | ⚡ | Quantum Engine 精度改进、汇率服务、性能优化 |
| Bug 修复 | 🐛 | 计算错误修复、信号误触发修复 |
| 基础设施 | 🔧 | 数据库、部署、监控、日志相关 |
| 文档更新 | 📝 | 策略文档、README、Skill 更新 |

## 输出格式

```markdown
# 策略变更日志 - v3.1.0 (2025-01-15)

## 📈 策略更新
- **Grid 策略斐波那契优化**: 新增 0.786 回撤位作为第五档网格节点
- **Trend Surfing 微调**: 降低强势行情下的回调等待阈值

## 🛡️ 风控调整
- **Risk Tier 3 上限调整**: 从 3.0% 调整为 2.5%，降低激进模式风险
- **回撤熔断通知**: 回撤达 15% 时触发预警（之前仅 20% 熔断）

## ⚡ 引擎优化
- **Quantum Engine**: 修复 JPY 跨币种汇率缓存失效问题
- **汇率服务**: 新增 crypto 对法币汇率的 fallback 路径

## 🐛 Bug 修复
- 修复 M15 收盘确认在跨天交易中的时区偏差
```

## 过滤规则

以下提交类型默认排除（不出现在变更日志中）：
- 纯重构 (Refactoring) 且无行为变更
- 测试文件更新
- IDE/编辑器配置变更
- merge commit
- CI/CD 配置调整
